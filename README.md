# English

## Chat App

Chat App is a real-time web messaging application that allows multiple users to communicate instantly through a shared server.
The main goal of this project was to learn how to build real-time applications using WebSockets with Socket.io and Express.

### How it works

- Users connect to the same Express server.
- Socket.io establishes a real-time WebSocket connection.
- Messages are emitted from the client and broadcast to all connected users.
- The chat displays system messages when users join the conversation.

### Built with

- HTML
- CSS
- JavaScript
- Node.js
- Express
- Socket.io

### Project purpose

This project was created to practice:

- Real-time communication
- WebSocket connections
- Server-client architecture
- Event-based programming
- Managing multiple users on a single server

## Requirements
Node.js (tested on v24.12.0, Windows)
Git (tested on v2.52.0, Windows)

## How to run the project
```bash
npm install
npm run start
```
